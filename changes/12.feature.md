Entity select menus are now fully supported. `MemberActor.select` accepts the handles a user could pick (`UserHandle`/`MemberActor`, `RoleHandle`, `ChannelHandle`) for user, role, channel and mentionable selects, building the resolved data so the bot's callback receives real `discord.Member`/`Role`/channel objects. Wrong handle kinds and out-of-range value counts fail with a `SetupError`.
