Added the final common-bot route sweep before the surface freezes for 1.0: `Guild.delete` (owner-only), `Member.fetch_voice` (read a member's voice state), `TextChannel.follow` (follow an announcement channel into a destination webhook), and `Guild.vanity_invite` (with `guild.set_vanity_url(code)` to populate it in the world builder). Routes whose result would have to be a constant empty value — integrations, welcome screen, widget, onboarding — are deliberately left as loud `RouteNotImplemented` gaps rather than faked; the parity matrix documents this as the frozen 1.0 gap surface.
